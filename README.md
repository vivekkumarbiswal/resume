## Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
ng serve
```

Open in browser:

```
http://localhost:4200
```
